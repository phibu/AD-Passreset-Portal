Env var name mapping (D-16 тАФ default `__` convention):
- `SmtpSettings:Password` тЖТ `SmtpSettings__Password`

Regression guard (Pitfall 5): `ClientSettings.Recaptcha.PrivateKey` has `[JsonIgnore]` so it never appears in the `GET /api/password` response, regardless of whether it was sourced from appsettings, user-secrets, or env var.
