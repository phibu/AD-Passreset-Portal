<objective>
STAB-015 + D-20 — Ship structured audit events via an allowlist DTO with RFC 5424 structured-data encoding. Compile-time redaction (no secret fields on the DTO). SD-ID configurable via `SiemSettings.Syslog.SdId` (default `passreset@32473`, validated).

Purpose: SIEM operators can index security events by outcome/traceId/user natively without log parsing, while compile-time DTO prevents accidental secret leaks.
Output: New `AuditEvent` record, extended `ISiemService`/`SiemService`/`SiemSyslogFormatter`, new `SdId` setting + validator rule, reflection test + formatter tests.
</objective>

         ```csharp
         /// <summary>STAB-015: RFC 5424 STRUCTURED-DATA emission for AuditEvent DTO.</summary>
         public static string Format(
             DateTimeOffset timestampUtc,
             int facility,
             int severity,
             string hostname,
             string appName,
             string sdId,
             AuditEvent evt)
         {
             var priority = (facility * 8) + severity;
             var ts = timestampUtc.ToUniversalTime().ToString("yyyy-MM-ddTHH:mm:ss.fffZ");
             var sd = new System.Text.StringBuilder();
             sd.Append('[').Append(sdId);
             sd.Append(" event=\"").Append(EscapeSd(evt.EventType.ToString())).Append('"');
             sd.Append(" outcome=\"").Append(EscapeSd(evt.Outcome)).Append('"');
             sd.Append(" user=\"").Append(EscapeSd(evt.Username)).Append('"');
             if (!string.IsNullOrEmpty(evt.ClientIp)) sd.Append(" ip=\"").Append(EscapeSd(evt.ClientIp)).Append('"');
             if (!string.IsNullOrEmpty(evt.TraceId))  sd.Append(" traceId=\"").Append(EscapeSd(evt.TraceId)).Append('"');
             if (!string.IsNullOrEmpty(evt.Detail))   sd.Append(" detail=\"").Append(EscapeSd(evt.Detail)).Append('"');
             sd.Append(']');
             return $"<{priority}>1 {ts} {hostname} {appName} - - - {sd}";
         }
         ```
